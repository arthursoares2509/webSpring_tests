\### O projeto está com seu modelo completo. 



Como ainda não tem backend definido, o projeto não irá rodar com precisão - apesar do teste da aplicação estar correto e o modelo rodar no servidor.



Para um teste completo do projeto, ao executar, utilize a coordenada "http://localhost:8080/filmes/index" para ter uma noção conceitual de como está ficando o projeto em si. 



AVISO: ao clicar no botão "Ver Filmes" do index, aparecerá uma página vazia contendo um erro de protocolo. Isso ocorre pois o projeto ainda é só um MODELO.

Dito isso, o esqueleto do projeto (XML), as classes modelo, o pacote de serviços e as páginas HTML de exemplos estão pronta. Ao rodar a aplicação, utilize apenas a coordenada dada acima.
